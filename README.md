# DDD CQRS demo

## Setup

Install Deno.

Run these commands to get started:

> Run the program

`deno run main.ts`

> Run the program and watch for file changes

`deno task dev`

> Run the tests

`deno test`

> Run the benchmarks

`deno bench`
